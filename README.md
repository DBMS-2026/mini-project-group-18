# 🎯 QueryQuest – Interactive DBMS Learning Platform

**QueryQuest** is a web-based platform designed to make learning Database Management Systems (DBMS) interactive, practical, and engaging through gamified experiences.

It combines SQL problem-solving, structured learning, and schema design into a single platform.

![Platform Demo](https://via.placeholder.com/800x400?text=QueryQuest+Demo)

---

## 🚀 Features

### 🕵️ SQL Noir
- Solve detective-style mystery cases using SQL queries  
- Each correct query unlocks clues and progresses the story  
- Immersive narrative-driven learning experience  

### 🎮 DBMS Quest
- Level-based learning system  
- Covers essential SQL concepts:
  - `SELECT`
  - `WHERE`
  - `JOIN`
  - `GROUP BY`
- Progressive difficulty curve  

### 🏗️ Schema Design Playground
- Design real-world database schemas  
- Create tables, relationships, and constraints  
- Get automated feedback on your designs  

### 📊 Progress Tracking
- Track learning progress over time  
- Identify weak areas  
- Monitor performance metrics

## 🚦 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

---

## ⚙️ Installation & Setup

### 1. Clone Repository
```bash
git clone https://github.com/yourusername/queryquest.git
cd queryquest
```

### 2. Install Frontend
```bash
cd project-dev
npm install
```

### 3. Install Backend
```bash
cd backend
npm install
```

---

## ▶️ Run the Application

### Terminal 1 (Frontend)
```bash
cd project-dev
npm run dev
```

### Terminal 2 (Backend)
```bash
cd backend
node app.js
```
